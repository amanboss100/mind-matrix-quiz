# ⚡ MIND MATRIX v4.5

An interactive, hyper-focused procedural quiz engine and developer portfolio portal built entirely inside a single high-performance `index.html` file. Designed for instant zero-config deployment on Vercel via GitHub.

---

## 🚀 Key Features

* **Infinite Procedural Generator:** Dynamic mathematics equation solver and science weight matching algorithms ensure unique questions on every single load. No loops or repetitive question pools.
* **Aesthetic Mono Grid:** Features a pure black-and-white moving vector graph-paper canvas background that continuously shifts coordinates seamlessly in the background.
* **PC & Mobile Responsive Synth Audio:** Custom-engineered real-time HTML5 audio nodes that play sharp, lag-free sound effects on user interaction without relying on external `.mp3` assets.
* **Integrated Branding Cards:** Built-in massive high-contrast neon contact blocks connecting directly to the developer's core network channels.

---

## 🛠️ Project Stack

* **Markup:** HTML5 (Semantic Structure)
* **Styling:** Tailwind CSS v4 (Aesthetic Utilities)
* **Logic Engine:** Vanilla JavaScript (ES6+ Functional Math & Matrix Arrays)
* **Graphics:** HTML5 Canvas 2D Context Renderer
* **Audio FX:** Web Audio API Synth Generators

## 📦 Instant Vercel Deployment

1. Create a new repository on GitHub named `mind-matrix-quiz`.
2. Commit and push the `index.html` and this `README.md` file to the main branch.
3. Open your Vercel Dashboard, click **Add New > Project**, and import this repository.
4. Click **Deploy**. Your live matrix grid terminal is active in less than 30 seconds!
